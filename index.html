安卓常用库

发表于 2016-03-01

网络

  • volley 非官方镜像库
  • okhttp

图片加载

  • Picasso - 与 okhttp 同为 square 出品
  • Glide - 与 Picasso API 极为相似，性能方面更具优势，推荐使用

UI

  • CircularProgressView - MD(material design)风格的 loading view
  • MaterialDateTimePicker - MD风格的日期选择控件
  • PhotoView - 图片预览
  • material-dialogs - MD风格对话框，功能十分强大

其他

  • butterknife - view 注入框架，省去到处写 findViewById
  • EventBus - 用于各组件通信
